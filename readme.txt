Add WP Author Profile Widget with easy way.  


== Description ==

<ul>
	<li>Shows author gravatar, name, website, description and social icons</li>
	<li>Fully customizable to match your theme design (style, color, size and text options)</li>
	<li>Nice looking on desktop, laptop, tablet or mobile phones</li>
</ul>

== Installation ==

Upload the WP Author Profile Widget plugin, Activate it, Update your Profile then go to the admin menu-> Appearence-> Widget-> WP Author Profile Widget Drag & drop on sidebar Widget. 

You're done! :)
